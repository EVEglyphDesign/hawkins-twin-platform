## What lives here

This folder is the **Hawkins overlay** on the EVE Glyph canon. It is a copy of the existing EVE Glyph canon (framing, naming, people-naming) plus additional files that Tim Hawkins asks for on the Hawkins side.

The pattern is:

- Files copied from `/canon/` are the EVE Glyph baseline. They govern tone, framing, and language across all EVE Glyph work, including the Hawkins twin.
- Files added under `hawkins/canon/` that are **not** in `/canon/` are Hawkins-side additions. They apply only to Hawkins twin deliverables.
- Where a Hawkins-side file explicitly overrides an EVE Glyph baseline, the override is called out at the top of that file, in a line beginning `Overrides:`.

## Rule for adding files

Any file added to `hawkins/canon/` must:

1. State its origin at the top (copied from EVE Glyph, or Hawkins-side original).
2. Name its owner (Dany, Tim, or joint).
3. State the date filed.
4. If it overrides an EVE Glyph baseline, name the override on the first line of the body.
